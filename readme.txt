A plugin to activate and configure plugins automatically for local development and testing environments.

== Description ==

Activate Plugins on Local is designed for developers working on WordPress sites in local environments. It helps automatically activate and configure plugins (such as Rank Math Pro) for development and testing, bypassing unnecessary API checks and external service requirements. This plugin is not intended for use on production sites.

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. The plugin will automatically work in your local environment.

== Frequently Asked Questions ==

= Is this plugin safe for production? =
No. This plugin is intended for local development only and should not be used on live/production sites.

= What plugins does it support? =
Currently, it supports Rank Math Pro and can be extended for other plugins as needed.
