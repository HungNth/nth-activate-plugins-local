# Activate Plugins on Local

A WordPress plugin that helps activate and configure plugins in local development environments for development and testing purposes.

## Description

This plugin is designed to assist developers during WordPress development and testing in local environments. The plugin automatically configures and activates necessary plugins without requiring connections to external services.

## Features

- Automatically configure plugins in local environments
- Bypass unnecessary API authentication requirements during development
- Support configuration for Rank Math Pro
- Does not affect production environments

## System Requirements

- WordPress: 6.0 or higher
- PHP: 7.4 or higher

## Installation

1. Download the plugin and extract it to the `wp-content/plugins/` directory
2. Activate the plugin from the WordPress Admin Plugins page
3. The plugin will automatically work in local environments

## Important Notice

⚠️ **This plugin should ONLY be used in local development environments. DO NOT use in production environments.**
